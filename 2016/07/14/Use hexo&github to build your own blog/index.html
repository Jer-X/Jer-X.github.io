Use hexo&github to build your own blog

发表于 2016-07-14   |   分类于 随写

摘要: This post show me how to build my blog

  1. 申请github账号
  2. 在github上新建一个repository,名字必须为 >账号名.github.io;
  3. 先到node.js官网上下载一个node.js并安装
  4. 在现在git的命令行软件
  5. 在某个地方新建一个文件夹
  6. 打开git命令行,用npm命令安装hexo,命令为 npm i -g hexo
  7. 用 hexo -v 来检查是否安装成功
  8. 成功后则在此文件夹中打开git命令行,输入 hexo init
  9. 使用npm install安装默认插件
  10. 再使用npm install hexo-deployer-git –save来安装github插件
  11. 使用hexo generate命令生成web项目,使用hexo deploy命令发布你的web项目

  12. 找到目录下的_config.yml文件并修改一下代码,把Jer-X改为你自己的账户名

  13. 然后你就可以使用hexo deploy命令来发布你自己的博客

  14. 最后记得访问一下xxx.github.io去查看自己的博客
